1보도자료 초안 3분 작성약 3분

📌 이런 상황

다음 주 「반려동물 문화교실」 행사 보도자료를 오늘 안에 넘겨야 함. 손에 있는 건 기안문·계획서뿐.

✍️ 프롬프트 (경량 모델 / 웹검색 OFF)

당신은 용인시청 홍보담당 주무관입니다. [첨부: 행사 계획서]를 바탕으로 보도자료를 작성하세요. 형식은 제목 → 리드 1문단 → 본문 3문단 → 문의처, 분량 800자 내외, 공공기관 격식체로 쓰고, 일시·장소·대상은 계획서 수치를 그대로 쓰되 지어내지 마세요.
✅ 나오는 결과
  • 바로 배포 가능한 완성형 보도자료 1건
  • 이어서 "제목 5안 더 뽑아줘", "카드뉴스 6장으로" 확장

💡 꿀팁 — 계획서를 붙여넣으면 날짜·장소 오류가 확 줄어요. 숫자는 마지막에 원문과 한 번 더 대조!

2조례안 요약 + 타 지자체 비교표약 5분

📌 이런 상황

신규 조례안을 검토 중. 핵심을 빠르게 파악하고 다른 시·군은 어떤지 비교해 보고서에 넣어야 함.

✍️ 프롬프트 (상위 모델 / 웹검색 ON)

당신은 지방자치법에 능통한 행정 전문가입니다. [아래 조례안 붙여넣기]의 핵심을 목적·주요조항·시행일로 요약하고, 유사 조례를 둔 [수원시·성남시·화성시]와 우리 안(案)을 비교표로 정리하세요. 각 항목은 근거 조문 번호와 출처 URL을 함께 표기하세요.
✅ 나오는 결과
  • 3줄 요약 + 4개 시 비교표(항목·근거·출처)
  • 이어서 "상위법과 충돌 없는지 검토해줘"로 리스크 점검

💡 꿀팁 — 법령·조문은 웹검색 ON으로 켜고, 링크를 직접 열어 교차 확인 후 인용하세요.

3민원 답변서 톤 조정 (딱딱→공감)약 3분

📌 이런 상황

규정대로 쓴 답변이 너무 딱딱해 민원인이 더 화날 것 같음. 규정은 그대로, 말투만 부드럽게.

✍️ 프롬프트 (경량 모델)

아래는 민원 답변 초안입니다. 인용한 규정·조치사항은 그대로 유지하되, 공감 → 설명 → 대안 → 마무리 순서의 친절한 공감형 톤으로 다시 써주세요. 첫 문장은 반드시 민원인의 불편에 공감하는 인사로 시작하세요. [초안 붙여넣기]
✅ 나오는 결과
  • 규정 근거는 유지 + 말투만 부드러워진 답변
  • 이어서 "불가 요청이면 대안까지 넣어 거절 답변으로"

💡 꿀팁 — "중학생도 이해할 쉬운 말로"를 덧붙이면 행정용어가 자동으로 풀립니다.

4회의록 → 실행과제 목록 정리약 2분

📌 이런 상황

1시간 회의가 끝났고 손메모만 잔뜩. 누가·언제까지 뭘 하기로 했는지 표로 정리해 공유해야 함.

✍️ 프롬프트 (경량 모델)

다음 회의 메모를 ①안건 ②주요 논의 ③결정사항 ④실행과제로 정리하세요. 실행과제는 담당자·기한·우선순위(상/중/하) 표로 만들고, 담당·기한이 메모에 없으면 '미정'으로 표기하세요. [회의 메모 붙여넣기]
✅ 나오는 결과
  • 구조화된 회의록 + 실행과제 표(담당/기한/우선순위)
  • 이어서 "이 표를 팀 공유용 안내 문자로"까지 확장

💡 꿀팁 — '미정' 표기를 시키면 지어내기(환각)가 줄고, 빠진 정보가 한눈에 보여요.
