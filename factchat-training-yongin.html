Session ①

Opening — 설명 대신 "즉시 체험" 5분

슬라이드 설명 대신, 전원이 아래 프롬프트를 휴대폰으로 바로 입력합니다. 30초 만에 완성문이 나오는 것을 체감한 뒤 "오늘은 이걸 내 업무에 맞게 바꾸는 법을 배웁니다"로 연결합니다.

당신은 용인시청 총무과 주무관입니다. "여름철 냉방온도 준수 협조" 내부 공지문을 공문 격식체로 5문장 이내로 작성해줘.
🤖 FactChat 강점(근거 인용 · 한국어 최적화 · 프로젝트 메모리)은 결과물을 함께 보며 1분 내로 짚어줍니다.
Session ②

Part 1. 크레딧 절약 — 같은 작업, 모델만 바꿔보기 10분

사례 1 · 행사 안내문 20자 요약 기본 모델

  1. 강사가 화면에서 상위 모델로 실행 → 크레딧 소모량 표시
  2. 수강생 전원은 기본(경량) 모델로 동일 실행 → 소모량 비교 (약 1/3~1/5)
아래 행사 안내문을 시 홈페이지 배너용으로 20자 이내로 요약해줘. "용인시는 오는 7월 20일 오전 10시 용인미르스타디움에서 시민과 함께하는 제5회 용인시민 걷기대회를 개최합니다. 참가 신청은 7월 15일까지…"
📌 모델 선택 기준 30초 정리 — 오탈자 교정·요약·번역 → 기본 / 조례-상위법 충돌 검토·정책 분석 → 상위 / 이미지·PPT → Super Agent

사례 2 · 프로젝트 기능 1분 시연

"우리 팀 보도자료" 프로젝트를 만들고 자주 쓰는 역할 문장을 저장 → 매번 입력하지 않아도 같은 톤이 유지되는 것을 시연합니다.

Session ③

Part 2. 프롬프트 3원칙 — Before/After 페어 실습 15분

사례 3 · 나쁜 프롬프트를 직접 겪어보기 (2인 1조)

✕ A가 입력 (Before)"보도자료 하나 써줘."

→ 행사명·날짜를 AI가 임의로 지어내는 환각을 직접 확인
✓ B가 입력 (After)아래 좋은 프롬프트 사용

→ 두 결과를 옆 사람과 비교하며 차이를 체감
당신은 용인시청 홍보 주무관입니다. 아래 행사 계획을 바탕으로 제목·리드문·본문 3문단·문의처 형식, 800자, 공공기관 격식체로 보도자료를 작성하세요. 수치와 일시는 계획 내용 그대로 사용하고 임의로 추가하지 마세요. [행사 계획] 제5회 용인시민 걷기대회 / 7.20(월) 10시 / 용인미르스타디움 / 참가비 무료 / 선착순 2,000명 / 문의: 체육진흥과 031-000-0000

사례 4 · 근거 요구로 환각 차단 웹 검색 ON

「폐기물관리법」 제8조(투기 금지)의 내용을 알려줘. 국가법령정보센터(law.go.kr) 원문을 근거로 하고, 인용 조문의 출처 URL을 함께 표기해줘.
🔗 답변에 붙은 링크를 눌러 원문 대조까지 직접 해보게 합니다.

3원칙 정리(2분): ① 역할 부여 ② 맥락 제공(부서·목적·독자·분량·톤) ③ 근거 요구 — C.R.E.A.T.E. 6단계는 「1장 요약 가이드」로 안내

Session ④

Part 3. 부서별 시나리오 미션 6종 20분

본인 업무와 가장 가까운 미션 2개를 골라 [대괄호]만 바꿔 실행합니다. 프롬프트 모음 페이지 QR을 띄워 복사하게 하세요.

미션 1 · 문서작성 — 공문 초안 기본 모델

당신은 용인시청 [○○과] 주무관입니다. [수신처: 관내 경로당 35개소]에 발송할 "[폭염 대비 냉방기 점검 협조]" 공문 초안을 표준 공문 형식(수신·제목·본문·붙임)으로 작성해줘.

미션 2 · 법령/조례 — 상위법 충돌 검토 상위 모델 + 웹 검색

당신은 지방자치법에 능통한 행정 전문가입니다. "[용인시 ○○ 지원 조례안]"의 아래 조항이 [상위법: ○○법]과 충돌하는지 검토하고, 관련 법령 조항과 출처 URL을 표기해줘. [조항 내용 붙여넣기]

미션 3 · 민원 — 답변서 작성 기본 모델

당신은 용인시청 [○○과] 민원 담당자입니다. 아래 민원에 대해 공감 표현 → 처리 경과 → 향후 계획 순서로, 정중한 격식체 500자 이내 답변서를 작성해줘. [민원 내용: 아파트 앞 불법투기 쓰레기가 2주째 방치되어 있습니다…]

미션 4 · 보고서 — 1페이지 요약보고 상위 모델

당신은 [기획팀] 주무관입니다. 아래 자료를 국장 보고용 1페이지 요약보고(개요·현황·문제점·개선방안·건의사항)로 정리해줘. 개조식으로 작성해줘. [자료 붙여넣기]

미션 5 · 회의록 — 녹취 정리 기본 모델

아래 회의 메모를 회의록 형식(일시·참석자·안건·결정사항·향후 일정)으로 정리하고, 결정사항은 담당자·기한을 표로 만들어줘. [메모 붙여넣기]

미션 6 · 데이터 — 표 만들기·통계 해석 기본 모델

아래 민원 접수 현황 데이터를 유형별로 집계한 표로 만들고, 전월 대비 증감이 큰 상위 3개 유형과 시사점을 3줄로 요약해줘. [데이터 붙여넣기]
🎯 진행 팁 — 미션 1·3·5·6은 기본 모델, 미션 2·4는 상위 모델로 지정해 Part 1의 모델 선택 전략을 실전 반복시킵니다.
Session ⑤·⑥

결과 공유 & Closing 10분

⑤ 결과 공유 — "오늘 아낀 30분" (5분)

2~3명이 자기 결과물을 화면에 띄워 발표합니다. "이 작업을 원래 몇 분 걸려 했는지 vs 지금 몇 분 걸렸는지"를 말하게 하여 교육명("30분 단축")을 회수합니다.

⑥ Closing (5분)

  • 배포자료: 1장 요약 가이드 + 프롬프트 6종 별첨 (QR 안내)
  • 실천 과제: "내일 출근해서 오늘 미션 프롬프트 중 1개를 실제 업무에 적용"
  • 사내 문의 채널 · 후속 심화교육 안내
