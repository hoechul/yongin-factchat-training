2. 교육 설계 방향

사전 설문에서 드러난 직원 페인포인트를 "바로 해결"하는 실전형 커리큘럼으로 구성했습니다.

애로사항 41%

크레딧 소진

✚
모델 선택 전략 — 단순 작업은 경량 모델, 심층 분석만 상위 모델 사용. 크레딧 3~5배 절약을 실시간 시연합니다.
💡 이렇게 씁니다
경량 모델로 충분회의록 오탈자 교정 · 행사 안내문 20자 요약 · 영↔한 번역 같은 단순·반복 작업 상위 모델 권장조례안이 상위법과 충돌하는지 검토 · 정책 타당성 분석 보고서 같은 추론·판단 작업 예: 같은 "행사 안내문 요약"을 경량 모델로 처리하면 상위 모델 대비 크레딧을 약 1/3~1/5만 소모합니다.
🤖 FactChat에서 대화 입력창의 모델 선택에서 기본(경량) ↔ 상위 모델을 작업에 맞춰 전환하세요.
애로사항 30%

응답 정확성 부족

✚
웹 검색 + 근거 링크 확인법 — 법령정보센터 URL을 첨부하는 프롬프트로 신뢰할 수 있는 답변을 얻습니다.
💡 이렇게 씁니다
근거를 지정하면 답변의 신뢰도가 올라가고, 링크를 직접 열어 원문 대조까지 할 수 있습니다. 「폐기물관리법」 제8조(투기 금지)의 내용을 알려줘. 국가법령정보센터(law.go.kr) 원문을 근거로 하고, 인용한 조문의 출처 URL을 함께 표기해줘.
🤖 FactChat에서 웹 검색을 켠 상태로 질문하고, 답변에 붙은 링크를 눌러 조문 원문과 대조하면 환각(잘못된 인용)을 막을 수 있습니다.
애로사항 29%

행정 맥락 부족

✚
행정 페르소나 지정 프롬프트 — "당신은 용인시청 기획팀 주무관입니다…"로 행정 문서 톤에 맞춥니다.
💡 이렇게 씁니다
역할 없이"환경정화 캠페인 안내문 써줘" → 구어체·일반적인 홍보 문구가 나옵니다. 역할 지정아래처럼 쓰면 존댓말·격식체·행정 용어가 자동으로 반영됩니다. 당신은 용인시청 기획팀 주무관입니다. 시민에게 배포할 '환경정화 캠페인' 안내문을 공공기관 공문 격식체로, 300자 내외로 작성해줘.
🤖 FactChat에서 자주 쓰는 역할 문장은 프로젝트에 저장해두면 매번 입력하지 않아도 같은 톤을 유지할 수 있습니다.
애로사항 15%

프롬프트 작성 어려움

✚
C.R.E.A.T.E. 프레임워크 — Context · Role · Example · Action · Tone · Evaluation의 6단계로 누구나 좋은 질문을 만듭니다.
💡 6단계를 한 문장에 담으면
C · 배경
"○○조례 개정 관련"
R · 역할
"기획팀 주무관"
E · 예시
기존 문서 붙여넣기
A · 지시
"3문단으로 작성"
T · 톤
"공문 격식체"
E · 검증
"근거·출처 표기"
당신은 용인시청 기획팀 주무관입니다.(R) 첨부한 회의 내용을 바탕으로(C·E) 보도자료 초안을 3문단·800자 내외로 작성하되(A), 공공기관 격식체로 쓰고(T), 인용 통계는 출처 URL을 함께 표기해줘.(E)
🤖 FactChat에서 배포자료 「1장 요약 가이드」의 C.R.E.A.T.E. 순서를 그대로 따라 입력하면 됩니다.
Curriculum · 60min

3. 교육 커리큘럼

Opening부터 실습·Q&A까지, 물 흐르듯 이어지는 60분 구성입니다.

①
Opening — 왜 FactChat인가
5분
  • 타 생성형 AI 대비 강점: 근거 인용 · 한국어 최적화 · 프로젝트 메모리
②
Part 1. 크레딧을 아끼는 스마트한 사용법10분
  • 모델 선택 가이드: 요약·번역·정리 → 기본 / 정책 분석·보고서 → 상위 / 이미지·PPT → Super Agent
  • 긴 대화 관리: 프로젝트 기능으로 대화 분리, 컨텍스트 재사용
  • 실패한 프롬프트 재활용법 (수정 재생성 vs 새 대화)

👉 자세한 모델 선택·크레딧 절약 팁은 FactChat 1장 요약 가이드에서 확인하세요.

③
Part 2. 정확도를 높이는 프롬프트 3원칙15분
  • 역할 부여 — "당신은 지방자치법에 능통한 행정 전문가입니다"
  • 맥락 제공 — 부서·목적·독자·분량·톤 명시
  • 근거 요구 — "관련 법령 조항과 출처 URL을 함께 표기하세요"
🎯 미니 실습 · 나쁜 프롬프트 vs 좋은 프롬프트
✕ 나쁜 프롬프트
"보도자료 하나 써줘."
나오는 응답주제·대상·수치가 없어 일반론적인 빈칸투성이 초안. 행사명·일시·장소를 AI가 임의로 지어내(환각) 결국 처음부터 다시 써야 함.
✓ 좋은 프롬프트
당신은 용인시청 홍보 주무관입니다. [행사 계획서]를 바탕으로 제목·리드·본문 3문단·문의처 형식, 800자, 공공기관 격식체로 보도자료를 작성하세요. 수치는 계획서 그대로.
나오는 응답날짜·장소가 정확하고 격식체를 지킨 바로 배포 가능한 완성형 보도자료. 역할·맥락·근거를 지정해 수정 없이 그대로 활용.

📱 개인 스마트폰 실습

Closing — Q&A 및 배포자료 안내
5분

Hands-on · 20min

행정 업무 시나리오 실습

교육이 끝나면 바로 쓸 수 있도록, 실제 행정 상황 4가지를 상황 → 복사용 프롬프트 → 나오는 결과까지 그대로 따라 합니다.

실습 1 · 보도자료
보도자료 초안 3분 작성
약 3분경량 모델 · 웹검색 OFF
📌 이런 상황
다음 주 「반려동물 문화교실」 행사 보도자료를 오늘 안에 넘겨야 함. 손에 있는 건 기안문·계획서뿐.
✍️ 복사용 프롬프트
당신은 용인시청 홍보담당 주무관입니다. [첨부: 행사 계획서]를 바탕으로 보도자료를 작성하세요. 형식은 제목 → 리드 1문단 → 본문 3문단 → 문의처, 분량 800자 내외, 공공기관 격식체로 쓰고, 일시·장소·대상은 계획서 수치를 그대로 쓰되 지어내지 마세요.
✅ 나오는 결과
  • 바로 배포 가능한 완성형 보도자료 1건
  • 이어서 "제목 5안 더 뽑아줘", "카드뉴스 6장으로" 확장
💡 꿀팁 — 계획서를 붙여넣으면 날짜·장소 오류가 확 줄어요. 숫자는 마지막에 원문과 한 번 더 대조!
실습 2 · 조례·법령
조례안 요약 + 타 지자체 비교표
약 5분상위 모델 · 웹검색 ON
📌 이런 상황
신규 조례안을 검토 중. 핵심을 빠르게 파악하고 다른 시·군은 어떤지 비교해 보고서에 넣어야 함.
✍️ 복사용 프롬프트
당신은 지방자치법에 능통한 행정 전문가입니다. [아래 조례안 붙여넣기]의 핵심을 목적·주요조항·시행일로 요약하고, 유사 조례를 둔 [수원시·성남시·화성시]와 우리 안(案)을 비교표로 정리하세요. 각 항목은 근거 조문 번호와 출처 URL을 함께 표기하세요.
✅ 나오는 결과
  • 3줄 요약 + 4개 시 비교표(항목·근거·출처)
  • 이어서 "상위법과 충돌 없는지 검토해줘"로 리스크 점검
💡 꿀팁 — 법령·조문은 웹검색 ON으로 켜고, 링크를 직접 열어 교차 확인 후 인용하세요.
실습 3 · 민원 응대
민원 답변서 톤 조정 (딱딱→공감)
약 3분경량 모델
📌 이런 상황
규정대로 쓴 답변이 너무 딱딱해 민원인이 더 화날 것 같음. 규정은 그대로, 말투만 부드럽게.
✍️ 복사용 프롬프트
아래는 민원 답변 초안입니다. 인용한 규정·조치사항은 그대로 유지하되, 공감 → 설명 → 대안 → 마무리 순서의 친절한 공감형 톤으로 다시 써주세요. 첫 문장은 반드시 민원인의 불편에 공감하는 인사로 시작하세요. [초안 붙여넣기]
✅ 나오는 결과
  • 규정 근거는 유지 + 말투만 부드러워진 답변
  • 이어서 "불가 요청이면 대안까지 넣어 거절 답변으로"
💡 꿀팁 — "중학생도 이해할 쉬운 말로"를 덧붙이면 행정용어가 자동으로 풀립니다.
실습 4 · 회의·정리
회의록 → 실행과제 목록 정리
약 2분경량 모델
📌 이런 상황
1시간 회의가 끝났고 손메모만 잔뜩. 누가·언제까지 뭘 하기로 했는지 표로 정리해 공유해야 함.
✍️ 복사용 프롬프트
다음 회의 메모를 ①안건 ②주요 논의 ③결정사항 ④실행과제로 정리하세요. 실행과제는 담당자·기한·우선순위(상/중/하) 표로 만들고, 담당·기한이 메모에 없으면 '미정'으로 표기하세요. [회의 메모 붙여넣기]
✅ 나오는 결과
  • 구조화된 회의록 + 실행과제 표(담당/기한/우선순위)
  • 이어서 "이 표를 팀 공유용 안내 문자로"까지 확장
💡 꿀팁 — '미정' 표기를 시키면 지어내기(환각)가 줄고, 빠진 정보가 한눈에 보여요.
🚀 어떤 시나리오든 통하는 3분 실습 순서
자료 준비부터 검수까지, 이 4단계만 기억하세요.
1
자료 준비
계획서·회의메모·초안을 손에 두기 (붙여넣거나 첨부)
2
모델 선택
단순정리=경량 / 분석·법령=상위 / 웹검색 필요 시 ON
3
프롬프트 입력
역할·형식·분량·톤·근거를 한 번에 지정
4
검수 후 활용
수치·법령·날짜 원문 대조 → 담당자 최종 확인

파란 상자의 프롬프트를 그대로 복사해 붙여넣고, [대괄호]와 노란 글씨(첨부·붙여넣기) 부분만 실제 자료로 바꾸세요.
