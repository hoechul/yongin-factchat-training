2. 교육 설계 방향

사전 설문에서 드러난 직원 페인포인트를 "바로 해결"하는 실전형 커리큘럼으로 구성했습니다.

애로사항 41%

크레딧 소진

✚
모델 선택 전략 + 프로젝트 활용 — 단순 작업은 경량 모델, 심층 분석만 상위 모델 사용. 자주 쓰는 역할 문장은 프로젝트에 저장해 반복 입력을 없앱니다. 크레딧 3~5배 절약을 실시간 시연합니다.
💡 이렇게 씁니다
경량 모델로 충분회의록 오탈자 교정 · 행사 안내문 20자 요약 · 영↔한 번역 같은 단순·반복 작업 상위 모델 권장조례안이 상위법과 충돌하는지 검토 · 정책 타당성 분석 보고서 같은 추론·판단 작업
📊 사례 1. 행사 안내문 20자 요약 경량 모델로 충분한 작업
강사가 화면에 아래를 상위 모델로 실행 → 크레딧 소모량 표시 수강생 전원은 기본(경량) 모델로 동일 실행 → 소모량 비교 (약 1/3~1/5)
📋 실습 프롬프트 · 복사 → 기본 모델 선택 후 입력
아래 행사 안내문을 시 홈페이지 배너용으로 20자 이내로 요약해줘. "용인시는 오는 7월 20일 오전 10시 용인미르스타디움에서 시민과 함께하는 제5회 용인시민 걷기대회를 개최합니다. 참가 신청은 7월 15일까지…"
📁 꿀팁. 프로젝트 기능으로 "역할 문장" 저장 반복 입력 제거 · 톤 유지
1) 「우리 팀 보도자료」 프로젝트를 새로 만들고, 자주 쓰는 역할·톤 지시 문장을 프로젝트 지침에 저장 2) 이후에는 역할 문장을 매번 다시 입력하지 않고 본문 자료만 붙여넣어도 같은 톤·형식이 자동 유지 3) 긴 지시를 반복 입력할 필요가 없어 입력량↓ · 재작성↓ → 크레딧 절약
📋 프로젝트 지침에 저장할 역할 문장 · 복사 → 프로젝트 설정에 붙여넣기
당신은 용인시청 기획팀 주무관입니다. 모든 답변은 공공기관 보도자료 격식체로, 제목·리드문·본문 3문단·문의처 형식, 800자 내외로 작성하고, 인용한 통계·수치는 출처 URL을 함께 표기하세요.
🤖 FactChat에서 좌측 프로젝트에서 새 프로젝트를 만들고 지침(Instructions)에 위 문장을 저장하면, 그 프로젝트 안 모든 대화에 자동 적용됩니다.
🤖 FactChat에서 대화 입력창의 모델 선택에서 기본(경량) ↔ 상위 모델을 작업에 맞춰 전환하세요.
애로사항 30%

응답 정확성 부족

✚
웹 검색 + 근거 링크 확인법 — 법령정보센터 URL을 첨부하는 프롬프트로 신뢰할 수 있는 답변을 얻습니다.
💡 이렇게 씁니다
근거를 지정하면 답변의 신뢰도가 올라가고, 링크를 직접 열어 원문 대조까지 할 수 있습니다. 「폐기물관리법」 제8조(투기 금지)의 내용을 알려줘. 국가법령정보센터(law.go.kr) 원문을 근거로 하고, 인용한 조문의 출처 URL을 함께 표기해줘.
🤖 FactChat에서 웹 검색을 켠 상태로 질문하고, 답변에 붙은 링크를 눌러 조문 원문과 대조하면 환각(잘못된 인용)을 막을 수 있습니다.
애로사항 29%

행정 맥락 부족

✚
행정 페르소나 지정 프롬프트 — "당신은 용인시청 기획팀 주무관입니다…"로 행정 문서 톤에 맞춥니다.
💡 이렇게 씁니다
역할 없이"환경정화 캠페인 안내문 써줘" → 구어체·일반적인 홍보 문구가 나옵니다. 역할 지정아래처럼 쓰면 존댓말·격식체·행정 용어가 자동으로 반영됩니다. 당신은 용인시청 기획팀 주무관입니다. 시민에게 배포할 '환경정화 캠페인' 안내문을 공공기관 공문 격식체로, 300자 내외로 작성해줘.
🤖 FactChat에서 자주 쓰는 역할 문장은 프로젝트에 저장해두면 매번 입력하지 않아도 같은 톤을 유지할 수 있습니다.
애로사항 15%

프롬프트 작성 어려움

✚
C.R.E.A.T.E. 프레임워크 — Context · Role · Example · Action · Tone · Evaluation의 6단계로 누구나 좋은 질문을 만듭니다.
💡 6단계를 한 문장에 담으면
C · 배경
"○○조례 개정 관련"
R · 역할
"기획팀 주무관"
E · 예시
기존 문서 붙여넣기
A · 지시
"3문단으로 작성"
T · 톤
"공문 격식체"
E · 검증
"근거·출처 표기"
당신은 용인시청 기획팀 주무관입니다.(R) 첨부한 회의 내용을 바탕으로(C·E) 보도자료 초안을 3문단·800자 내외로 작성하되(A), 공공기관 격식체로 쓰고(T), 인용 통계는 출처 URL을 함께 표기해줘.(E)
🤖 FactChat에서 배포자료 「1장 요약 가이드」의 C.R.E.A.T.E. 순서를 그대로 따라 입력하면 됩니다.

3. 교육 커리큘럼

①
Opening — 왜 FactChat인가5분
  • 타 생성형 AI 대비 강점: 근거 인용 · 한국어 최적화 · 프로젝트 메모리
②
Part 1. 크레딧을 아끼는 스마트한 사용법10분
  • 모델 선택 가이드: 요약·번역·정리 → 기본 / 정책 분석·보고서 → 상위 / 이미지·PPT → Super Agent
  • 긴 대화 관리: 프로젝트 기능으로 대화 분리, 컨텍스트 재사용
  • 실패한 프롬프트 재활용법 (수정 재생성 vs 새 대화)

👉 자세한 모델 선택·크레딧 절약 팁은 FactChat 1장 요약 가이드에서 확인하세요.

③
Part 2. 정확도를 높이는 프롬프트 3원칙15분
  • 역할 부여 — "당신은 지방자치법에 능통한 행정 전문가입니다"
  • 맥락 제공 — 부서·목적·독자·분량·톤 명시
  • 근거 요구 — "관련 법령 조항과 출처 URL을 함께 표기하세요"
🎯 미니 실습 · 나쁜 프롬프트 vs 좋은 프롬프트
✕ 나쁜 프롬프트
"보도자료 하나 써줘."
나오는 응답주제·대상·수치가 없어 일반론적인 빈칸투성이 초안. 행사명·일시·장소를 AI가 임의로 지어내(환각) 결국 처음부터 다시 써야 함.
✓ 좋은 프롬프트
당신은 용인시청 홍보 주무관입니다. [행사 계획서]를 바탕으로 제목·리드·본문 3문단·문의처 형식, 800자, 공공기관 격식체로 보도자료를 작성하세요. 수치는 계획서 그대로.
나오는 응답날짜·장소가 정확하고 격식체를 지킨 바로 배포 가능한 완성형 보도자료. 역할·맥락·근거를 지정해 수정 없이 그대로 활용.
④
Part 3. 행정 업무 시나리오 실습20분
각자 휴대폰으로 실제 업무를 그대로 재현하는 핵심 세션입니다. 실습 미션 페이지 열기 ↗
