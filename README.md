# 용인시 전 직원 대상 FactChat 활용 역량 강화 교육 — 랜딩페이지

행정 업무, AI로 30분 단축하기. 용인시청 임직원 대상 FactChat 실무 활용 역량 강화 교육 기획안을 소개하는 단일 랜딩페이지입니다.

## 구성

- **교육 개요** — 대상, 일시(2026. 7. 7.), 시간(60분), 방식(오프라인 + 실습)
- **교육 설계 방향** — 사전 설문 페인포인트(크레딧·정확성·행정맥락·프롬프트) → 해결 방안
- **커리큘럼** — Opening → Part 1~3 → Closing 60분 타임라인
- **행정 업무 시나리오 실습** — 보도자료 / 조례 요약 / 민원 답변 / 회의록 정리
- **배포자료** — 1장 요약 가이드, 행정 프롬프트 30선

## 기술 스택

의존성 없는 정적 사이트 (단일 `index.html`, 임베디드 CSS/JS). 빌드 단계 없이 Vercel에 바로 배포됩니다.

## 로컬 미리보기

```bash
# 아무 정적 서버나 사용
npx serve .
# 또는 브라우저로 index.html 직접 열기
```

## 배포

Vercel 정적 배포 (`vercel --prod`).
